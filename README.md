# Reports

## Overview
This repository contains cybersecurity analysis reports covering multiple areas of security operations, including phishing investigations, log analysis, OSINT, and SOC-style incident investigations.

The purpose of this project is to build practical experience in security analysis and develop foundational SOC analyst skills.

---

## What I Practice
- Phishing email analysis and social engineering detection  
- Email header analysis 
- Log analysis and interpretation (system, network, security logs)  
- SOC alert investigation and incident triage  
- OSINT investigations  
- Basic threat detection and incident reporting  
- Mapping findings to MITRE ATT&CK framework  

---

## Repository Content

- `reports/` → Cybersecurity analysis reports (phishing, logs, OSINT, SOC cases, etc.)  
- `screenshots/` → Evidence and supporting material (emails, logs, alerts, images)

Each report represents a separate investigation scenario.
